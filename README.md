# Moontasir Fuad Plabon - Portfolio

AI Integration & QA Automation Specialist portfolio website.

## View Live

Open `index.html` in any browser, or deploy to any static hosting (GitHub Pages, Netlify, Vercel).

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to Settings → Pages → Source: `master` branch
3. Site will be live at `username.github.io/portfolio`

## Tech Stack

- Single HTML file (vanilla HTML/CSS/JS)
- No dependencies, no build step
- Dark theme with coral accent
- Responsive design
- Google Fonts (Inter, JetBrains Mono)
- Smooth scroll animations

## Sections

- Hero with name, title, CTAs
- About with stats
- Skills grid (AI, Automation, API, DevOps)
- Projects showcase (6 projects)
- Work experience timeline
- Certifications
- Contact info
